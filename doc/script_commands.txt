* DebugLog(message)
Writes a debug message to the console

// Dialog Commands
* dlg_title("<title>")
Sets the title of a dialog

* dlg_text("<text>")
Sets the text of a dialog

* dlg_menu("<menu text>", "<menu function>")
Sets a menu option to the dialog

* dlg_show()
Sends the dialog to the player

// 
* get_value("<value name>")
Gets a value from the attached character
